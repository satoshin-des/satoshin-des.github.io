使ってる言語について

取り敢えずgithubのプロフィールのやつを引っ張ってきましょう。

で、どの言語をいつ使ってるの？

言語 いつ使う？
C 最近はあんまり使わない。
C++ 研究で使う。NTLライブラリとかEigenライブラリとか数学に関するライブラリが充実してて神。
最近はゲーム開発でもめちゃくちゃ使う。まぁ、DXライブラリありきだけど。
C# ゲーム開発で使う。Unity使いやすいけどまだまだ使いこなせてない。
emojin もはやお遊びでしか使わない。これを研究で使っている人いたら教えてください。
HSP ゲーム開発で使う。めちゃくちゃ使いやすい。まじ神。
もうHSPから離れられないかも知れない。
HTML 言うまでもなく、こういうHPなんかを作るときに使ってる。
LaTeX 論文とか数学系の同人誌とか書くとなったらLaTeXしかない。
でもTeXを[tex]みたいに発音するならpythonも[python]とか[pyθon]みたいに発音したらどうだ。
Python 色々と使うことが多い。研究でも使うことがあるし、お遊びで使うこともある。
あと競プロでも使うことがしばしば。遅いのがネック。
Risa/Asir たまーに使う。Groebner基底まわりを研究してる友達はよく使ってるイメージ。
弊学に開発者がいるのも大きいのかも。
SageMath 研究で使うことが多い。有理数が扱いやすいのはもちろん、行列とかイデアルとか格子とか、兎に角数学の道具が使いやすい神。
